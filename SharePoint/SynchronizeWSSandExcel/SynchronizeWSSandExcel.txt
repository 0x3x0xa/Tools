To Add the Extension to Excel:

- Open Excel Client

- Go to File --> Options --> Add-Ins

- Select 'Go' button and browse to the 'XL2007SynchronizeWSSandExcel' file to add this to Add-Ins

Tip: Remember to save your Excel files in the earlier version .xls file format!
