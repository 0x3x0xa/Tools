Créez une entrée pour stocker vos sessions à distance, vos identifiants, vos données, vos contacts et bien plus.

Nous ajouterons une nouvelle Session à distance.

La session RDP est maintenant créée. Dans l’onglet Général, entrez un nom pour votre session, le nom de l’ordinateur distant, le nom d’utilisateur et le mot de passe.

Sélectionnez votre session, puis cliquez sur Ouvrir la session.
